Introduction to Wargames

Wargames are a simulation of an armed conflict or war with the most realism possible from the strategic perspective. Typically with 2 teams against each other in a scenario with the aim to defeat the other team using superior strategy,

How It Works

Typically, the host will give both sides a version of the map. This will have everything from that side on it, including units, fortifications and other structures that side has built. Then, each side will begin to give orders for their units. Once complete, the host will determine the result based on a simulated dice roll and various other factors such as terrain, unit stats, morale, etc.

After the result of the order is determined, the maps will be updated with any new relevant information, and the result will be told to one or both sides depending on the action. Usually, each side will have one or more objectives and the completion of them will be the win condition for that team.

While this is the basic setup of a wargame, the exact specifics can change depending on the wargame, but generally it will more-or-less follow the same format.

Making your own

If you are familiar with wargames you may want to host one yourself. For that, first you should make a map. While you can use any software for this, the easist and simplest way is to use Google My Maps. Upon creating a new map, first you should create the borders and territory with the line tool

After creating the territory of the two sides, you can start on the units, which are done by using markers, and giving them a custom icon.

After you're finished creating the map, you can start the game. To change a unit's stats once it takes damage, simply edit the text in the unit's description, while to move units, drag the icon to its new location.
